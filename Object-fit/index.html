object-fit: fill (default):

A imagem da uma esticada ele está sendo redimensionado independente da proporção

Imagem original:

Resolução: 960x640, Tamanho : 140x140

object-fit: contain:

Imagem será redimensionada com sua proporção mantida para que a imagem inteira caiba,dentro do contêiner. portando sobra espaço no container

Imagem original:

Resolução: 960x640, Tamanho : 140x140

object-fit: none:

a imagem não será redimensionada, de forma a manter suas dimensões originais. Mas, se ela for menor que o contêiner, a imagem inteira será exibida com seu tamanho original. Já se for maior que o contêiner, ela será cortada.

object-fit: cover:

a imagem não será redimensionada, de forma a manter suas dimensões originais. Mas, se ela for menor que o contêiner, a imagem inteira será exibida com seu tamanho original. Já se for maior que o contêiner, ela será cortada.
